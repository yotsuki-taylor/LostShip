# Lost Ship — Стимпанк выживание

Веб-игра на React + Tailwind CSS: выживание летающего стимпанк корабля. Тёмная тема в стиле терминала управления.

## Запуск

```bash
npm install
npm run dev
```

Откройте в браузере адрес, который покажет Vite (обычно http://localhost:5173).

## Интерфейс

- **Сверху** — панель ресурсов: Hull (корпус), Energy (энергия), Scrap (лом), Crew (экипаж), Stability (стабильность).
- **Центр** — лог событий и блок «Текущее решение» с описанием события и двумя вариантами выбора.
- **Справа** — модули корабля: список систем с кнопками улучшения (трата лома).

## Механика

- Каждое событие предлагает два варианта; выбор меняет ресурсы согласно заданным дельтам.
- После выбора тикает ход: восстанавливается энергия, случайно выбирается новое событие.
- Улучшения модулей повышают максимальные значения ресурсов (корпус, энергия, лом, экипаж, стабильность).
